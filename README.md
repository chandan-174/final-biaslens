# AI Data & Decision Intelligence System

## Overview

An academic-oriented dashboard for data analysis and decision intelligence:

- Google authentication (Supabase)
- CSV upload + KPI summaries + charts
- Bias signal detection with severity + suggested mitigations
- AI chat + document scanning for bias-aware reasoning support

## Local development

If you want to work locally using your own IDE, you can clone this repo and push changes.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Install dependencies.
npm i

# Start the dev server.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Academic note

Outputs are decision support, not ground truth. Validate findings with domain context and appropriate statistical methods.
